Compiling on Linux

  1. You need gcc installed
  2. Install SDL2, SDL2_ttf and SDL2_mixer using Your distro's package manager
  3. Clone the repo, compile and install:
    4. 					
git clone https://github.com/jszczerbinsky/KOEngine
cd KOEngine
./build.sh
cd build
sudo cp -R * /usr

Compiling on Windows (MinGW)

  1. You need MinGW or mingw-w64 installed
  2. Download and install the following development libraries for mingw: SDL2, SDL2_ttf and SDL2_mixer
  3. Clone the repo:
    4. 					
git clone https://github.com/jszczerbinsky/KOEngine
cd KOEngine
explorer .
  4. Run build.bat
  5. Go to build directory
  6. Copy directories into Your compiler's instance path (the same path where You installed previous development libraries)
