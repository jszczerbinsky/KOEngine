Setting up project

Create main.c file and include KOE/KOE.h: 
				
#include <KOE/KOE.h>

int main(int argc, char* argv[])
{
}
Now define 2 functions - onStart and loopCall: 
				
void onStart()
{
}

void loopCall()
{
}
The first one will be called right after KOEngine initialization. It should be used to load game resources and also to initialize a game state (load first level, spawn player etc.). The second one will be called on every tick of the loop timer. It should be used to update a game state while game is running (moving player, killing enemies etc.).
After creating these functions You need to initialize KOEngine using struct KOEngineSettings and KOEngineInit function: 
				
const struct KOEngineSettings engineSettings = {
  .windowName = "My window",
  .onStartPtr = &onStart,
  .loopCallPtr = &loopCall,
  .onMusicEndPtr = NULL,
  .audioChannels = 0
};

KOEngineInit(&engineSettings);

Compiling Your project

If You are using Linux, You should compile Your project with the following flags: 
-lKOEngine -lSDL2 -lSDL2_ttf -lSDL2_mixer
However if You are using Window, use these instead: 
-lmingw32 -lKOEngine -lSDL2 -lSDL2_ttf -lSDL2_mixer
Example: 
gcc [Your .c files] -lmingw32 -lKOEngine -lSDL2 -lSDL2_ttf -lSDL2_mixer -o [Your output file].exe

Running your project

Note, that on Windows You have to put KOEngine, SDL2, SDL2_ttf and SDL2_mixer runtime libraries in your output .exe file directory

You should now be able to run the game and see a blank screen.
